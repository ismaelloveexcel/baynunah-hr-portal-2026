# AI Agents Configuration

This directory contains configuration files for all 8 AI agents in the multi-agent architecture.

## Agent Overview

1. **Supervisor Agent** - Orchestrates all agents and workflow
2. **Discovery Agent** - Scans repositories and builds feature inventory
3. **Solution Architect** - Creates proposals and architecture
4. **Builder Agent** - Rebuilds and merges features
5. **UI Excellence Agent** - Enforces visual quality standards
6. **UX Reviewer Agent** - Ensures great user experience
7. **Product Gap Analyst** - Identifies missing features
8. **Azure Deploy Assist** - Provisions and deploys to Azure

## Configuration Structure

Each agent configuration file includes:
- Agent name and role
- Responsibilities
- Input requirements
- Output format
- Azure OpenAI model settings
- Integration points with other agents
